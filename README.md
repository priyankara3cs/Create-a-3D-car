- React Three

## Server Links

### vercel

- https://create-a-3-d-car.vercel.app/

---

## Run react in development mode

### npm install

```sh
npm install --force
```

### Development Server

Start the development server on `http://localhost:3000`:

```sh
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```
